This readme will always contain a description of the other folders in the repo. 

adodb5: The library needed to have PHP code that can interact with Access database files

dropdown: The set of code that creates dynamic dropdowns based on databases.

jsTree: The jQuery tree library we are attempting to use for having a tree view of the ANZSIC data.  

treeTute: Experimental code for creating trees using the Tree module provided by nicmart. 

treeDatabase: Experimental code for creating a tree based of the database ANZSIC codes.
